# Utils

#### 概述
	一个通用的js 工具类
	

#### 参考
https://jofunliang.github.io/usuallyjs/


#### 安装方式
     <script src="./utils.min.js"></script>
     import utils from "@neotrident/utils"
    

#### 示例

   ```javascript
<template>
<div id="app">
    
</div>
</template>

<script>
import utils from "@neotrident/utils"

export default {
    name: 'app',
    mounted(){
        console.log(utils)
    }
}
</script>
```
